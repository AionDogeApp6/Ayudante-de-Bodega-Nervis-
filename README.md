
# 📦 Ayudante de Bodega Nervis

**Ayudante de Bodega Nervis** es una aplicación web ligera y minimalista diseñada para emprendedores y dueños de negocios que necesitan un control de inventario rápido, preciso y sin complicaciones.

Esta herramienta permite gestionar ventas multimoneda, controlar existencias de productos detallados (por peso o volumen) y mantener la privacidad total de los datos.

---

## ✨ Características Principales

*   **🧮 Cálculo Multimoneda:** Integración automática con la tasa oficial del **BCV** para convertir ventas de USD a Bolívares al instante.
*   **⚖️ Control de Detallados:** Soporte para productos vendidos por **Unidad, Kilogramos (g) y Litros (ml)**. Ideal para charcutería, víveres y líquidos.
*   **📂 Sistema de Respaldos:** Botones de "Copia de Seguridad" para descargar tu inventario en un archivo JSON y cargarlo en otros dispositivos (PC o móvil).
*   **🔍 Buscador Inteligente:** Filtra tus productos rápidamente por nombre.
*   **📱 Diseño Responsivo:** Funciona perfectamente en navegadores de PC y puede instalarse como una "Web App" en el celular.
*   **🔒 Privacidad Total:** Los datos se almacenan localmente en el navegador del usuario (LocalStorage). No se requiere base de datos externa ni registro.

---

## 🚀 Cómo usar esta herramienta

1.  **Carga de Inventario:** Abre el panel de gestión (icono de engranaje) y registra tus productos con su precio y stock inicial.
2.  **Venta Diaria:** Ingresa la cantidad vendida en la fila correspondiente. El sistema calculará el total en la parte inferior.
3.  **Confirmación:** Al presionar "Confirmar Venta", el stock se descontará automáticamente del inventario.
4.  **Respaldo:** Al finalizar el día o la semana, descarga tu respaldo para asegurar tus datos.

---

## 🛠️ Tecnologías utilizadas

*   **HTML5 / CSS3:** Interfaz limpia y optimizada para móviles.
*   **JavaScript (Vanilla):** Lógica de cálculo y gestión de estado.
*   **DolarAPI:** Para la obtención de la tasa cambiaria en tiempo real.
